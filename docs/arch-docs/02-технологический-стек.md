# Технологический стек

## Frontend

| Компонент  | Технология  | Назначение                     |
| ---------- | ----------- | ------------------------------ |
| HTML/CSS   | HTML5, CSS3 | Структура и стили интерфейса   |
| JavaScript | ES6 modules | Бизнес-логика клиентской части |
| Three.js   | r128        | 3D-визуализация (WebGL)        |

### Модули JavaScript

- `js/core/` — конфигурация и константы
- `js/ui/` — управление интерфейсом
- `js/services/` — валидация параметров
- `js/bridge/` — взаимодействие с Python
- `js/utils/` — вспомогательные функции

## Python-среда

| Компонент    | Технология | Назначение                    |
| ------------ | ---------- | ----------------------------- |
| Pyodide      | 0.26.0     | Python WebAssembly в браузере |
| ifcopenshell | 0.8.4      | Создание и обработка IFC      |

### Особенности Pyodide

- Загрузка ~80 МБ при старте
- Установка зависимостей: `typing_extensions` → `numpy` → `shapely` → `ifcopenshell`
- Виртуальная файловая система для модулей

## IFC-стандарт

| Параметр       | Значение                                |
| -------------- | --------------------------------------- |
| Стандарт       | IFC4 ADD2 TC1 (version 4.0.2.1)         |
| Класс сущности | IfcMechanicalFastener                   |
| Тип геометрии  | IfcSweptDiskSolid, IfcExtrudedAreaSolid |
| Представление  | Body (MODEL_VIEW)                       |

### Единицы измерения

- Длина: миллиметры (мм)
- Масса: килограммы (кг)
- Угол: радианы

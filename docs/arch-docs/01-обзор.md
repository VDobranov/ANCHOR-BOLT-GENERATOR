# Обзор проекта

## Назначение

**ANCHOR-BOLT-GENERATOR** — браузерное приложение для параметрического создания фундаментных (анкерных) болтов по **ГОСТ 24379.1-2012** с экспортом в формат **IFC4 ADD2 TC1** и **3D-визуализацией** в браузере.

## Целевая аудитория

- Инженеры-проектировщики фундаментов
- BIM-менеджеры
- Специалисты по металлоконструкциям

## Основные возможности

| Возможность                   | Описание                                     |
| ----------------------------- | -------------------------------------------- |
| Параметрическое моделирование | Выбор типа, диаметра, длины болта по ГОСТ    |
| 3D-визуализация               | Интерактивный просмотр в браузере (Three.js) |
| IFC-экспорт                   | Генерация файлов для BIM-процессов           |
| Валидация                     | Проверка параметров по ГОСТ 24379.1-2012     |
| Работа в браузере             | Без установки ПО (Python через WebAssembly)  |

## Поддерживаемые типы болтов

| Тип | Форма           | Диаметры | Длины    |
| --- | --------------- | -------- | -------- |
| 1.1 | Изогнутый       | М12–М48  | 300–1400 |
| 1.2 | Изогнутый       | М12–М48  | 300–1000 |
| 2.1 | Прямой с плитой | М16–М48  | 500–1250 |
| 5   | Прямой          | М12–М48  | 300–1000 |

## Состав сборки

Автоматически определяется по типу болта:

- **Типы 1.1, 1.2, 5**: шпилька + шайба + 2 гайки
- **Тип 2.1**: шпилька + анкерная плита + шайба + 4 гайки

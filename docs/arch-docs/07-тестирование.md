### Mock-классы

Для изоляции от ifcopenshell.geom:

```python
class MockIfcEntity:
    def __init__(self, entity_type, **kwargs):
        self._entity_type = entity_type

    def is_a(self):
        return self._entity_type

class MockIfcDoc:
    def __init__(self):
        self.entities = []

    def create_entity(self, entity_type, **kwargs):
        entity = MockIfcEntity(entity_type, **kwargs)
        self.entities.append(entity)
        return entity

    def by_type(self, entity_type):
        return [e for e in self.entities if e.is_a() == entity_type]
```

### Запуск тестов

```bash

pytest tests/test_geometry_builder.py -v
```

## Валидация IFC (Behave + Gherkin)

### Назначение

Проверка сгенерированных IFC-моделей на соответствие спецификации IFC4.

### Правила buildingSMART

Используются правила из **ifc-gherkin-rules** (buildingSMART):

| Категория | Описание | Примеры |
|-----------|----------|---------|
| IFC | Версия IFC, deprecated сущности | IFC101, IFC102 |
| MPD | RepresentationType/Identifier | MPD001 |
| GEM | Геометрическая валидация | GEM001, GEM003 |
| SPS | Пространственная структура | SPS001, SPS007 |

### Конфигурация

`tests/features/environment.py`:

```python
def before_all(context):
    # Путь к step definitions
    gherkin_rules_path = project_root / "external" / "ifc-gherkin-rules" / "features" / "steps"
    sys.path.insert(0, str(gherkin_rules_path))
```

### Пример правила (Gherkin)

```gherkin
Feature: IFC101 - Only official IFC versions allowed

  Scenario: Verifying Current Schema Identifier
    Given An IFC model
    Then The Schema Identifier of the model must be 'IFC2X3' or 'IFC4' or 'IFC4X3_ADD2'
```

### Запуск валидации

```bash
# Запуск Behave
behave tests/features/

# Конкретное правило
behave tests/features/rules/IFC/IFC101*.feature
```

## CI/CD интеграция

### GitHub Actions

```yaml
name: Tests
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.13'
      - name: Run tests
        run: pytest tests/ -v
```

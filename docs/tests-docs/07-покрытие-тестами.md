## Отчёт о покрытии

### Python coverage

```bash
# Установка coverage
pip install pytest-cov

# Запуск с coverage
pytest tests/ --cov=python --cov-report=term-missing

# HTML отчёт
pytest tests/ --cov=python --cov-report=html
open htmlcov/index.html  # macOS
```

### JavaScript coverage

```bash
# Запуск с coverage
npm run test:coverage

# Отчёт в coverage/
open coverage/index.html
```

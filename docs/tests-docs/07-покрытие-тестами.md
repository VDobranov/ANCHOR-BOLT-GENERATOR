## Отчёт о покрытии

### Python coverage

```bash
# Установка coverage
pip install pytest-cov

# Запуск с coverage
pytest tests/ --cov=python --cov-report=term-missing

# HTML отчёт
pytest tests/ --cov=python --cov-report=html
open htmlcov/index.html  # macOS
```

### JavaScript coverage

```bash
# Запуск с coverage
npm run test:coverage

# Отчёт в coverage/
open coverage/index.html
```

## Исключение модулей из coverage

В `package.json` заданы исключения для сложных модулей:

```json
"collectCoverageFrom": [
  "js/**/*.js",
  "!js/tests/**",
  "!js/form.js",
  "!js/viewer.js",
  "!js/ifcBridge.js",
  "!js/main.js",
  "!js/init.js",
  "!js/ui.js",
  "!js/config.js"
]
```

Это позволяет поддерживать высокий порог покрытия (80%) для тестируемых модулей, исключая модули, требующие сложной mock-инфраструктуры.

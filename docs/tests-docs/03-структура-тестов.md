## Gherkin валидация

```
tests/features/
└── environment.py              # Конфигурация Behave

external/ifc-gherkin-rules/
└── features/rules/             # 100+ правил валидации

    ├── GEM/                    # Геометрическая валидация
    ├── SPS/                    # Пространственная структура
    └── ...
```

## Mock классы (conftest.py)

| Класс | Назначение |
|-------|------------|
| `MockIfcEntity` | Mock для IFC сущности |
| `MockIfcDoc` | Mock для IFC документа |

## Фикстуры (conftest.py)

| Фикстура | Описание |
|----------|----------|
| `mock_ifc_doc` | Создание Mock IFC документа |
| `valid_bolt_params` | Параметры валидного болта |
| `all_bolt_types` | Все поддерживаемые типы болтов |
| `all_diameters` | Все доступные диаметры |
| `all_materials` | Все доступные материалы |

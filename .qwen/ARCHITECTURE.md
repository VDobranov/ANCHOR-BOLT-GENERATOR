# ANCHOR-BOLT-GENERATOR — Архитектура

## Обзор

**Генератор анкерных болтов** — браузерное приложение для создания фундаментных болтов по **ГОСТ 24379.1-2012** с экспортом в **IFC4** и **3D-визуализацией**.

→ Подробнее: [docs/arch-docs/01-обзор.md](../docs/arch-docs/01-обзор.md)

## Технологический стек

| Компонент      | Технология                       |
| -------------- | -------------------------------- |
| Frontend       | Vanilla JS (ES6 modules)         |
| 3D-движок      | Three.js r128                    |
| Python-среда   | Pyodide 0.26.0 (WebAssembly)     |
| IFC-библиотека | IfcOpenShell 0.8.4               |
| Тестирование   | pytest + Jest + Behave (Gherkin) |

→ Подробнее: [docs/arch-docs/02-технологический-стек.md](../docs/arch-docs/02-технологический-стек.md)

## Поток данных

1. **Ввод параметров** → Валидация (JS + Python)
2. **TypeFactory** → Получение/создание типа с кэшированием
3. **InstanceFactory** → Создание инстансов (IfcMappedItem)
4. **GeometryBuilder** → Построение IFC-геометрии
5. **GeometryConverter** → Конвертация IFC → mesh Three.js
6. **Viewer** → Обновление 3D-сцены

→ Подробнее: [docs/arch-docs/05-поток-данных.md](../docs/arch-docs/05-поток-данных.md)

## Ключевые решения

- **Единый источник истины** — геометрия строится в IFC, конвертируется в mesh
- **RepresentationMaps + IfcMappedItem** — переиспользование геометрии типов
- **Динамическое кэширование** — оптимизация размера IFC
- **Работа в браузере** — Python через Pyodide WebAssembly

→ Подробнее: [docs/arch-docs/06-ключевые-решения.md](../docs/arch-docs/06-ключевые-решения.md)

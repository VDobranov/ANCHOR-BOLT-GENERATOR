```markdown
- 2 верхних гайки
- 2 нижних гайки

- Z=0: начало резьбы (реперная точка)

- Нижняя гайка 2: самая нижняя
- Нижняя гайка 1: над плитой

**Диапазоны размеров:**
- Диаметры: М16, М20, М24, М30, М36, М42, М48
- Длины: 200–2800 мм
```

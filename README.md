# ANCHOR-BOLT-GENERATOR

Генератор анкерных болтов (фундаментных болтов) по ГОСТ 24379.1-2012 с экспортом в IFC и 3D визуализацией в браузере.

## Быстрый старт

### Запуск

```bash
python3 -m http.server 8000
```

Откройте в браузере: **http://localhost:8000**

### Использование

1. Выберите тип болта (1.1, 1.2, 2.1 или 5)
2. Выберите диаметр (М12–М48)
3. Выберите длину (из доступных по ГОСТ)
4. Выберите материал (09Г2С, ВСт3пс2 или 10Г2)
5. Настройте параметры экспорта IFC (опционально)
6. Вращайте 3D-вид мышью (ЛКМ — вращение, колесо — зум)
7. Нажмите «Скачать IFC» для экспорта

**Состав сборки определяется автоматически:**
- Типы 1.1, 1.2, 5: шпилька + шайба + 2 гайки
- Тип 2.1: шпилька + **анкерная плита** + шайба + 4 гайки

## Настройки экспорта IFC

### Класс сборки

Определяет тип IFC-сущности для болта:

- **IfcMechanicalFastener** (по умолчанию) — стандартный класс для крепежа
- **IfcElementAssembly** — класс для сборок элементов

### Режим формирования

Определяет способ представления геометрии:

- **Вроссыпь** (по умолчанию) — каждый компонент (шпилька, гайки, шайба, плита) создаётся как отдельный объект IFC
- **Всё одним телом** — все компоненты объединяются в один объект с помощью булевых операций

### Тип геометрии

Определяет способ представления 3D-геометрии:

- **Твердотельная** (по умолчанию) — стандартная геометрия через `IfcExtrudedAreaSolid`, `IfcSweptDiskSolid`, `IfcCSGSolid`
- **Поверхностная (faceted)** — полигональная геометрия через `IfcFacetedBrep` (конвертация solid в mesh)

## Параметры болтов

### Типы и размеры

| Тип и исполнение | Диаметры | Длины | Состав сборки |
|-----------------|----------|-------|---------------|
| 1.1 | М12–М48 | 300–2800 | Шпилька + шайба + 2 гайки |
| 1.2 | М12–М48 | 300–2800 | Шпилька + шайба + 2 гайки |
| 2.1 | М16–М48 | 200–2800 | Шпилька + **анкерная плита** + шайба + 4 гайки |

**Примечание:** Доступные длины зависят от диаметра и типа болта. Полный список генерируется автоматически из ГОСТ данных.

### Система координат

**Реперная точка: Z=0 — низ резьбы**

**Позиционирование компонентов:**

| Тип | Позиция шпильки | Компоненты |
|-----|----------------|------------|
| 1.1, 1.2 | Z=0 до Z=+L | Выше Z=0: шайба, 2 гайки |
| 2.1 | Z=-(L-l₀) до Z=+l₀ | Выше Z=0: шайба, 2 гайки<br>Ниже Z=0: шпилька, гайка, плита, гайка |
| 5 | Z=-(L-l₀) до Z=+l₀ | Выше Z=0: шайба, 2 гайки |

**Обозначения:**
- **L** — полная длина болта
- **l₀** — длина резьбы

**Пример для М20x300 тип 2.1:**
- l₀ = 100 мм, L = 300 мм
- Шпилька: от Z=-200 до Z=+100
- Z=0: низ резьбы
- Нижние гайки и плита: под Z=0

## Технологии

- **Three.js r128** — 3D визуализация в браузере
